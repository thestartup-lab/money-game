## 1. 遊戲概述與流程

### 遊戲階段

| 階段 | 說明 |
|------|------|
| 等待玩家 | 等待玩家加入房間 |
| Pre-20 | 20 歲前成長點數分配、職業選擇 |
| 老鼠賽跑 | 內圈主要遊戲歷程 |
| 財務自由外圈 | 脫出老鼠賽跑後進入的外圈 |
| 事件處理中 | 系統暫時狀態 |
| 遊戲結束 | 觸發最終結算與排名 |

### 房間系統
- 房間代碼：6 字元英數（排除易混淆字元 0、O、I、1）
- 已結束的房間禁止新玩家加入

## 2. 時鐘驅動年齡系統

### 核心概念
遊戲年齡不靠回合計算，**純粹由真實時間驅動**。遊戲開始後，現實時間的流逝直接對應玩家虛擬年齡的增長。

### 年齡設計
- 遊戲開始年齡：**20 歲**
- 遊戲結束年齡：**100 歲**
- 年齡跨度：80 年

### 遊戲時長設定
| 類型 | 時長 |
|------|------|
| 預設 | 90 分鐘 |
| 最短（快速場） | 60 分鐘 |
| 最長（長場） | 120 分鐘 |

### 暫停機制
- 主持人或結構性節點可暫停時鐘
- 暫停期間年齡**凍結**，累積暫停時間從遊戲總時長中扣除

### 人生階段劃分

| 階段 | 年齡區間 | 危機頻率係數 | 薪資倍率 |
|------|----------|-------------|---------|
| 青年（打拼期） | 20–34 歲 | × 1.0 | × 1.0 |
| 成家期 | 35–49 歲 | × 1.2 | × 1.0 |
| 轉型期 | 50–64 歲 | × 1.5 | × 1.0 |
| 退休期 | 65–79 歲 | × 1.8 | × 0.5 |
| 傳承期 | 80–100 歲 | × 2.2 | × 0.0（無薪資，靠被動收入） |

## 4. 職業系統

### ESBI 四象限分類概述

- **E 象限（受薪族）**：固定月薪，有固定支出，班表固定
- **S 象限（自僱者）**：薪資浮動（隨機/人脈驅動/技能驅動），部分有彈性行程
- **B 象限（企業主）**：薪資極低或為零，收入來自起始事業資產現金流
- **I 象限（投資者）**：無薪資，收入來自起始投資組合股息，初始財商值較高

## 7. 發薪日系統

### 觸發時機
每次移動路徑**路過或停在**發薪日格時觸發。一次移動可路過多個發薪日格，每個都會結算。

### 月淨現金流計算公式

```
月淨現金流 = 薪資 + （被動收入 × 財商乘數 × 外圈乘數）+ 婚姻月加成 − 月總支出
```

- 外圈乘數：進入外圈後被動收入 × **2.0**，內圈為 × 1.0
- 婚姻月加成：依婚姻類型不同（詳見婚姻系統）

### 月總支出組成

```
月總支出 = 稅金（年度計算）+ 房貸 + 車貸 + 信用卡 + 其他支出 + 保險月費 + 子女月費
```

- 每個孩子月費：$500
- 保險月費依持有種類加總（詳見保險系統）

### 發薪日規劃流程
1. 套用發薪日規劃投資（扣款並更新屬性）
2. 現金 += 當月淨現金流
3. 健康值自然衰退（可由投資阻止）
4. 人脈值自然成長
5. 年度繳稅（每 4 個發薪日觸發一次）

**決策時間上限：30 秒**，逾時自動略過所有投資。

### 發薪日規劃投資選項

| 投資項目 | 費用 | 效果 |
|---------|------|------|
| 升級財商值 | 依當前等級（見 FQ 升級費用表） | FQ +1 |
| 維護健康 | $200 | 阻止本次 HP 自然衰退 |
| 積極投資健康 | $500 | HP +20，且同時阻止本次衰退 |
| 進修培訓 | $600 | SK +20 |
| 主動拓展人脈 | $400 | NT +1（上限 10） |
| 股票定期定額 | $1,000 / $2,000 / $5,000 | 累積指數股票基金 |
| 購買保險 | 啟動費（見保險系統） | 啟動對應保險 |

### 股票定期定額（DCA）
- 可投入金額三選一：$1,000 / $2,000 / $5,000，或不投入
- 模擬月報酬率：**0.5%**（複利）
- 重複投入時累加至同一「指數股票基金」資產

### 人脈值自然成長
- 每 **2 個發薪日**自動 NT +1（上限 10）

### 裁員期間薪資
- 裁員發薪日剩餘次數 > 0 時：**薪資計為 $0**，每次發薪遞減 1

### 旅遊薪資懲罰
- 旅遊後下次發薪日薪資乘以 **0.7**（七折）

## 8. 稅務系統

### 年度繳稅觸發
- 每 **4 個發薪日**（= 遊戲一年）結算一次
- 年收入 = 當月總收入 × 4

### 累進稅率級距

| 應稅收入 | 稅率 |
|---------|------|
| $0 – $50,000 | 5% |
| $50,001 – $120,000 | 12% |
| $120,001 – $250,000 | 20% |
| $250,001 – $500,000 | 30% |
| $500,001 以上 | 40% |

### 年度節稅扣除額

| 扣除項目 | 每年扣除額 | 條件 |
|---------|---------|------|
| 每位撫養子女 | $15,000 / 人 | 每個孩子 |
| 醫療保險 | $2,400 | 持有醫療險 |
| 壽險 | $1,200 | 持有壽險 |
| 財產 / 企業險 | $3,600 | 持有財產險 |

### 計算流程
```
應稅收入 = max( 0, 年收入 − 所有扣除額加總 )
稅金 = 對應稅收入套用累進稅率
```

- 若現金不足，現金可能降為負數（代表欠稅）

| 保險類型 | 月保費 | 首次購買啟動費 | 年度節稅扣除 | 保護對象 |

### 保險功能詳述

- **醫療保險**：觸發醫療類危機（如心臟病、重大疾病）時費用豁免或大幅降低
- **壽險**：觸發壽險類危機（嚴重交通意外、罕見疾病）時費用豁免；玩家死亡時計算遺產忽略所有負債
- **財產／企業險**：觸發財產類危機（火災、天災、訴訟）時費用豁免

### 取消保險
- 可隨時取消，無退費，立即停止月扣保費

在外圈發薪日格（0、6、12）觸發：
```
紅利現金 = 持有所有資產的總市場估值 × 1%
```

## 20. 死亡機制

### 三種死亡觸發路徑

#### 路徑 A：危機卡致死（內圈）
- 危機卡具有致死能力 + 未持有對應保險 + 現金不足支付費用
- 廣播最終評分與淘汰通知給全房

#### 路徑 B：外圈疾病危機致死
- 落在外圈「疾病危機」格，觸發疾病危機牌，條件同路徑 A
- 廣播玩家死亡通知

#### 路徑 C：臥床自然死亡
- HP 降至 0 → 進入臥床狀態
- 每回合輪到該玩家擲骰時：**30% 機率自然死亡**
- 未死亡則跳過本回合（廣播回合跳過通知）
- 死亡後廣播最終評分與淘汰通知

### 死亡後處理流程
1. 標記為已死亡，從回合順序中移除
2. 計算死亡時的最終人生評分
3. 廣播評分結果與淘汰事件給全房
4. 玩家無法再行動，但評分仍納入最終排名

## 22. 人生評分 Life Score

### 7 個原始維度（各 0–100 分）

| 維度 | 計算公式 | 滿分條件 |
|------|---------|---------|
| 淨資產 | min(100, max(0, 淨資產 ÷ 1000)) | 淨資產 $100,000 |
| 被動收入 | min(100, 年化被動收入 ÷ 500) | 年被動收入 $50,000 |

| 傳承 | 見下方特殊規則 | 遺產淨值 $100,000 |

### 傳承分特殊規則

| 情況 | 計算方式 |
|------|---------|
| 無子女 | 固定 **50 分**（中性，無後代責任） |
| 有子女 + 持有壽險 | 遺產 = 資產總市值 + 現金（負債被壽險清償） |

| 遺產 ≥ $100,000 | 傳承分 = 100 |

## 23. 勝負條件

### 全場結束條件
**虛擬年齡達到 100 歲**時，遊戲立即進入結束狀態，觸發最終結算。

### 最終排名依據
依**幸福總分（0–100 分）由高到低排名**，分數相同時無額外決勝機制。

### 個人出局
符合死亡觸發條件的玩家**立即停止行動**，但其最終評分仍納入結束時的總排名。

### 無傳統「全員淘汰」機制
- 玩家死亡不會提前結束遊戲（遊戲持續至 100 歲）
- 死亡玩家的評分會在結算時一併顯示於排行榜
